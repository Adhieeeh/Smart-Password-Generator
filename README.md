# Smart Password Generator 🔐

A simple yet effective tool to generate secure, random passwords based on user preferences.

## Features
- **Custom Length:** Choose exactly how long your password needs to be.
- **Character Control:** Toggle symbols and numbers on or off.
- **Instant Generation:** Uses Python's `random` module for quick results.

## Concepts Learned
- Using Python's **Standard Libraries** (`random` and `string`).
- **For Loops** to construct strings character by character.
- **Function Parameters** to pass user preferences into logic.
- **Type Casting** (turning input strings into integers).
